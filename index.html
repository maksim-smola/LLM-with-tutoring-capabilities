О проекте

Цель проекта — показать, как ИИ может поддерживать обучение

Педагогический подход

Ассистент не ограничивается короткой подсказкой: он объясняет ход решения и задает вопрос, который помогает ученику подумать самому.

Четкая структура

Ответы строятся по единому формату: объяснение, шаг решения и вопрос для продолжения работы.

Учебная направленность

Проект подходит для демонстрации возможностей языковых моделей в школьном и студенческом обучении.

Архитектура

Как устроен сайт

Frontend

HTML, CSS и JavaScript формируют интерфейс, а также отправляют вопрос на сервер без перезагрузки страницы.

Backend

Flask принимает запрос `/ask`, проверяет данные и обращается к API языковой модели.

Model API

Cerebras API генерирует ответ с учетом системной инструкции образовательного тьютора.

Разработано в рамках исследовательского проекта по образовательным нейросетям.

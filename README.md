LLM with tutoring capabilities is an experimental educational language model designed to simulate the behavior of an adaptive tutoring system.
The project explores how a general-purpose LLM can be guided to provide structured explanations, step-by-step reasoning, and personalized learning support comparable to digital tutoring environments.

The model is configured to:

explain concepts progressively from basic to advanced levels
adapt responses to learner needs
support school-level STEM and humanities topics
provide structured reasoning rather than short answers
simulate tutoring-style interaction instead of standard chatbot responses

The goal of the project is to evaluate whether prompt-based alignment and instructional structuring can approximate tutoring-system adaptivity without additional model training.

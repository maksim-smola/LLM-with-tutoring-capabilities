О проекте

Цель проекта - показать, как ИИ может поддерживать обучение

Педагогический подход

Ассистент объясняет ход решения и задает вопрос, который помогает ученику подумать самостоятельно.

Четкая структура

Ответы строятся по единому формату: объяснение, шаг решения и вопрос для продолжения работы.

Учебная направленность

Проект подходит для демонстрации возможностей языковых моделей в школьном и студенческом обучении.

Возможности

Функции, которые делают сайт полезнее

Отдельный чат

Диалоговый интерфейс вынесен на отдельную страницу, чтобы главная оставалась обзорной и легкой.

Формулы

MathJax отображает математические выражения в удобном виде, включая LaTeX.

Подсказки

Готовые примеры вопросов помогают быстро проверить работу ассистента.

Архитектура

Как устроен сайт

Frontend

HTML, CSS и JavaScript формируют интерфейс и отправляют вопрос на сервер без перезагрузки страницы.

Backend

Flask принимает запрос `/ask`, проверяет данные и обращается к API языковой модели.

Model API

LLM генерирует ответ в соответствии с заданными учебными инструкциями.

Разработано в рамках исследовательского проекта об использовании больших языковых моделей в роли интеллектуального помощника при обучении. Асмолова Максима
